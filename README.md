# Course Demand Prediction System

## Overview

This project predicts student enrollment for university courses using Machine Learning. It helps academic planners allocate resources efficiently.

## Features

* Synthetic data generation
* SQLite database storage
* Multiple ML models:

  * Linear Regression
  * Decision Tree
  * Random Forest
* Model comparison and selection
* Batch prediction
* Visualization dashboard

## Project Structure

```
course-demand-prediction/
│
├── train_pipeline.py
├── model_training.ipynb
├── dashboard.ipynb
├── models/
├── data/
└── README.md
```

## Technologies Used

* Python
* Pandas, NumPy
* Scikit-learn
* SQLite
* Matplotlib, Seaborn

## How to Run

1. Run `train_pipeline.py` to train models and save outputs.
2. Open `dashboard.ipynb` to view visual analysis.
